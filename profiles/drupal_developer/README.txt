About
=====

Drupal developer is a pack of modules and settings designed to help setting up a
simple development environment.

Features
========

* Default content types: page and story
* Create one test user on install.
* Give permission to authenticated users to switch users to help testing (!)
* Fast install by default values for email address, username and password:
  username: admin, password: drupal by default.


Developer notes
===============

Updating the makefile (for newest contribs for example):

drush convert-makefile drupal_developer.make drupal-org.make
